# Express Goodreads GetBooks API

A lightweight Node.js and Express backend service designed to fetch, manage, and query book data inspired by Goodreads.

---

## 🚀 Features

* **RESTful API Endpoints:** Clean endpoints to fetch book details, authors, and genres.
* **Database Integration:** Lightweight data persistence/querying setup.
* **Query Parameters:** Support for filtering, sorting, and searching books by title or author.
* **Error Handling:** Robust centralized middleware for smooth API error responses.

---

## 🛠️ Tech Stack

* **Backend:** Node.js, Express.js
* **Database:** SQLite 
* **Dependencies:** `dotenv`, `cors`, `nodemon` (development)
